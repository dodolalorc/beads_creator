# Beads Creator

一个用于制作拼豆图纸的本地化工具，支持图片量化、手动涂色、批量换色，以及导出 PNG、CSV、JSON。

## 功能

- 上传图片后按当前画布尺寸和选中色板生成拼豆图纸
- 手动修改像素颜色
- 批量替换颜色
- 导出统计信息和图纸数据
- 支持 Web 与 Tauri 桌面端运行

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```

## Tauri

```bash
npm run tauri dev
npm run tauri build
```
